## find
`find` — утилита для рекурсивного поиска файлов и директорий.
```bash
# Поиск по имени
find . -name "wassup.txt"

# Поиск конфигов в /etc изменённых за последние 2 дня
find /etc -type f -mtime -2
```

Поддерживает фильтрацию по типу (`-type`), пустым файлам (`-empty`),
времени изменения (`-mtime`, `-cmin`) и времени доступа (`-atime`).

## Потоки вывода
Каждый процесс в Linux имеет три стандартных потока:

| Поток  | Номер | Описание        |
|--------|-------|-----------------|
| stdin  | 0     | ввод            |
| stdout | 1     | вывод           |
| stderr | 2     | вывод ошибок    |
```bash
# Перезаписать файл выводом
find . -type f -size -1G > result.txt

# Добавить в конец файла
find . -type f -size -1G >> result.txt

# Отправить ошибки в никуда
find . -type f -size +1G 2>/dev/null
```

## grep
`grep` — поиск текста внутри файлов.
```bash
# Рекурсивный поиск целого слова
grep -r -w 'golang' .
```

`-r` — рекурсивно, `-w` — целое слово, а не его часть.
